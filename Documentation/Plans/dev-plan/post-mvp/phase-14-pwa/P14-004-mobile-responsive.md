# P14-004: Mobile-Responsive Wizard UI

## Description
Optimize wizard interface for mobile screens (minimum iPhone 11 Pro 5.8").

## Acceptance Criteria
- [ ] Responsive wizard layout (stack vertically on mobile)
- [ ] Touch-friendly button sizes (min 44px)

- [ ] Test on iOS and Android browsers
