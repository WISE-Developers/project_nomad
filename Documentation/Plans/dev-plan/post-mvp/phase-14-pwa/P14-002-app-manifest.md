# P14-002: App Manifest and Icons

## Description
Create PWA manifest and app icons for installation.

## Acceptance Criteria
- [ ] Create manifest.json with app metadata
- [ ] Generate icon set (multiple sizes for different devices)
- [ ] Set theme color matching fire theme
- [ ] Set display mode to standalone
- [ ] Configure start URL

## Dependencies
- Phase 0 (Frontend setup)

## Estimated Time
2 hours

## Files to Create/Modify
- `frontend/public/manifest.json`
- `frontend/public/icons/` (icon-192.png, icon-512.png, etc.)
- `frontend/index.html` (manifest link)

## Notes
- Use existing nomad.svg as base for icons
- Icons should look good on both light/dark backgrounds
